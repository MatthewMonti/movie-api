Objective

This application helps you sort through movie database and save your most charished film.

User Logic URL HTTP Method Query Parameters Data Format Response to format
Welcome Message / GET / String Welcome Message HTML with Image
Login to User and Movie database Access /login POST Parameters Username and Password Session Token Created after authication
Full Movie List A-Z /movies GET /movies Login Token Auth + String List of Movies A-Z
Film by Title /movies/title/:title GET :title Login Token Auth + String A JSON object of a single film
Film by Release Year /movies/release/:releaseyear GET :releaseyear Login Token Auth + String A JSON object with matching release date
Age Appropriate films /movies/rated/:rated GET :rated Login Token Auth + String A JSON object with films matching age of audience
Rotten Tomatoes Ratting of Film /movies/rating/:rating GET :rating Login Token Auth + String A JSON object of films matching that rating
Genre of film by category /movies/genres/:genreName GET :genreName Login Token Auth + String A JSON object of specific films matching category
Film by Director /movies/directors/:name GET :name Login Token Auth + String A JSON object of films made by specific director
User Guide of APP /about_app/documentation GET /documentation String A webpage on how to use app
User see all users /users GET Login Token Auth + String A JSON object of login in user, structured like: {"Username": "", "Password": "", then after login token created where all users displayed }
User access to other User Account info /users/:Username GET :Username Login Token Auth + String A JSON object of login in user, structured like: {"Username": "", "Password": "", then after login token created where user look up by Username}
User is created /users POST Login -> Session Token -> Body Raw JSON String A JSON object of a new user, structured like: {"Username": "", "Password": "", "Email": ""} A JSON object of the new user added, structured like: {"Username": "","Password": "", "Email":""}
User change information /users/:Username PUT :Username Login -> Session Token -> Body Raw JSON String A JSON object where the user info edited
Allows user to delect account /users/:Username DELETE :Username JSON Raw String Text confirming user account delected
