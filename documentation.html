MyFlix App Purpose

This application helps you sort through movie database and save your most charished film.

User Logic URL HTTP Method Query Parameters Response to format
Welcome Message / GET / Welcome Message HTML with Image
Login to User and Movie database Access /login POST JSON object Username and Password User created or Username already exist
Favorite Movie Added /users/:Username/movies/:MovieID' POST JSON object Password with string of ObjectID of Movie Favorite movie displayed with User information
Favorite Movie Deleted /users/:Username/movies/:MovieID' DELETE JSON object Password with string of ObjectID of Movie Favorite movie removed displayed with User information
User Guide of APP /about_app/ GET String A webpage on how to use app
Full Movie List A-Z /movies GET /movies Login with Params Username & Password Login and Copy Bearer Token (User info) to Authenticate Bearer Token See a List of Movies A-Z
Film by Title /movies/title/:title GET :title Login with Params Username & Password Login and Copy Bearer Token (User info) to Authenticate Bearer Token type Movie Name after /title/ see movie info
Film by Release Year /movies/release/:releaseyear GET :releaseyear Login and Copy Bearer Token (User info) to Authenticate Bearer Token type release year after /releaseyear/ movies with date displayed
Age Appropriate films /movies/rated/:rated GET :rated Login and Copy Bearer Token (User info) to Authenticate Bearer Token type age rated after /rated/ movies that are age appropriate listed
Rotten Tomatoes Ratting of Film /movies/rating/:rating GET :rating Login and Copy Bearer Token (User info) to Authenticate Bearer Token type rating after /rating/ movies with that rating displayed
Genre of film by category /movies/genres/:genreName GET :genreName Login and Copy Bearer Token (User info) to Authenticate Bearer Token type category after /rating/ movies with that category displayed
Film by Director /movies/directors/:name GET :name Login and Copy Bearer Token (User info) to Authenticate Bearer Token type director name after /director/ movies with that director displayed
User see all users /users GET Login and Copy Bearer Token (User info) to Authenticate Bearer Token type /users/ ALL Users displayed
User access to other User Account info /users/:Username GET :Username Login and Copy Bearer Token (User info) to Authenticate Bearer Token type Username after /users/ info about that user displayed
User Registration /users POST A JSON object of a new user, structured like: {"Username": "", "Password": "", "Email": ""} Display New User info or message already exists
User change information /users/:Username PUT :Username Login and Copy Bearer Token (User info) to Authenticate Bearer Token type Username after /users/ update user info displayed The user can also update favorite movies
User can post Favorite Movie /users/:Username/movies/:MovieID' POST Login and Copy Bearer Token (User info) to Authenticate Bearer Token type username and favorite movie after /users/ new fav film displayed
User can delete Favorite Movie /users/:Username/movies/:MovieID DELETE Login and Copy Bearer Token (User info) to Authenticate Bearer Token type username and favorite movie after /users/ removed fav film displayed
Allows user to delect account /users/:Username DELETE :Username Text confirming user account delected
