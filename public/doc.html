MyFlix App Purpose

This application helps you sort through movie database and save your most charished film.

Usage of Command URL command Method of data extraction Data instructions JSON FORMAT Display message along with outcome of data change
Welcome Message / GET NONE NA Welcome Message
User Guide of APP /api/about_app/ GET NONE NA A webpage on how to use the app
Full Movie List A-Z /api//movies GET NONE See a List of Movies A-Z
Film by Title /api/movies/title/:title GET NONE NA See one movie by Title
Film by Release Year /api/movies/release/:releaseyear GET NONE NA A list of films by release year displayed
Age Appropriate films /api/movies/rated/:rated GET NONE NA Films filter by age appropreiate
Rotten Tomatoes Ratting of Film /api/movies/rating/:rating GET NONE NA List of films filter by Rotten Tomatoes Percentage
Genre of film by category /api/movies/genres/:genreName GET NONE NA Films filtered by category are displayed
Film by Director /api/movies/directors/:name GET NONE NA List of films filtered by director name
User see all users /api/users GET Login and Copy Bearer Token (User info) to Authenticate Bearer Token Only need to copy token authication once to access all database queries NA Full list of registered users
User Registration /api/users POST Follow the format below to Create User in body of raw JSON NOTE date is in decending order from large to small time block {"Username": "USERNAME", "Password": "PASSWORD", "Email": "EMAIL", "Birthday": "####-##-##", } Display New User info or message already exists
User change information /api/users/ PUT Login and Copy Bearer Token (User info) to Authenticate Bearer Token type Username after /users/ update user info displayed NOTE - date is in decending order from large to small time block { "Username": "USERNAME", "Password": "PASSWORD", "Email": "EMAIL", "Birthday": "####-##-##", } Display UPDATED User information
User can add Favorite Movie /api/:Username/Favorite POST Login and Copy Bearer Token (User info) to Authenticate Bearer Token then change URL to add fav movie URL before submitting put JSON object of favorite movie you want to POST. {"Favorite": "id"} Message that film has been added
User can delete Favorite Movie /api/:Username/Favorite DELETE Login and Copy Bearer Token (User info) to Authenticate Bearer Token then change URL to add fav movie URL before submitting put JSON object of favorite film you want to DELETE. {"Favorite": "id"} Message that film has been added
